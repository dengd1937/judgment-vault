# Knowledge Vault Design

这个目录保存知识库与 Claude Code skill 的设计草稿。

当前目标不是创建一个普通收藏夹，而是建立一个用于提升技术判断力的知识系统。系统重点沉淀：

- 技术选择背后的前提、约束和取舍。
- 来源材料的证据质量、适用边界和反例价值。
- 自己真实决策的上下文、拒绝方案和后续结果。
- 从多个案例中抽象出的可复用判断模式。
